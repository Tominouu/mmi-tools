Comment héberger son site Wordpress gratuitement ?

10 mai 2025 Développement Web 4 min de lecture

Dans cet article vous allez pouvoir apprendre à comment héberger votre site Wordpress gratuitement (en attendant d'avoir un hébergement plus robuste avec nom de domaine), vous allez également découvrir comment migrer votre local vers la version en ligne.

Étape 1: Création de l'environnement

Pour commencer vous allez devoir vous créer un compte sur ce site: Infinityfree

Cliquez sur le bouton "Register Now"


Si tout va bien, après avoir créé votre compte vous devriez obtenir cette page ci-dessous.

Dashboard de votre compte

Maintenant vous allez devoir cliquer sur le bouton "Create Account" en violet.


Cliquez sur le bouton "Create Now"

Vous allez maintenant devoir sélectionner le forfait que vous souhaitez utiliser, dans notre cas ça sera le tout premier.


Choisissez votre nom de domaine

Vous pouvez choisir votre nom de domaine, ainsi que l'extension (comment va se finir votre lien), vous êtes un peu limité dans le choix mais on ne va pas râler car tout est gratuit !


Cliquez sur le bouton "Create Account"

Maintenant vous allez noter votre mot de passe (il s'agit du mot de passe qui permet de vous connecter au serveur via FTP), cliquez sur l'œil pour révéler le mot de passe.
Ensuite, choisissez ce que vous voulez pour Email Consent, il n'y a rien d'obligatoire donc libre à vous.
Maintenant vous pouvez cliquer sur Create Account.


Attendez que Infinityfree setup votre site

Vous devez attendre que la page recharge, les configurations serveurs sont en train de se faire, ça prend généralement 3 minutes, ne vous inquiétez pas si ça prend plus longtemps.


Votre site est maintenant créé

Parfait votre site a maintenant été créé, vous devez tester si votre site a été configuré correctement.
Pour se faire copiez la ligne en dessous de "DOMAIN", pour mon cas il s'agit de "exemple-mmi.free.nf", et collez la dans votre navigateur
